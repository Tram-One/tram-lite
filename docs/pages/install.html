Install

Tram-Lite is easy to include in any project, and requires no build tooling!

script tag

To install, you can include a script tag pointed to unpkg.com in your index.html:

This will automatically add all the API methods for HTML attributes. This will also add the TramLite object to the window, which can be used to call the Javascript API.

types

If you use npm, you can get the type definitions by installing tram-lite, even if you don't use any build tooling (make sure this matches your script tag above).

This will automatically annotate Tram-Lite methods with documentation in your project if you are using a type-aware editor (like Visual Studio Code).
