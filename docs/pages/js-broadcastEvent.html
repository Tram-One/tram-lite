TramLite.broadcastEvent

TramLite.broadcastEvent is a function to dispatch events up or down the light and shadow DOM. This can be useful when you want other elements to react to an event happening from this element.

It is the underlying function behind , and tl-rebroadcast.

Syntax

Parameters

element

Element to associate the event with. It can be referenced using event.detail.originalElement. It is the reference point for which children or parent elements will receive the event.

eventName

The name of the event to dispatch.

eventDirection optional

The direction to send the event. Can be up, triggering on parent elements, or down, triggering on child elements. Note, when using down, the event target will be the child element - use event.detail.originalElement to have a reference to the original web-component. Defaults to up.

Return Value

None
