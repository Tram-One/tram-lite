ComponentDefinition

ComponentDefinition is a custom-element that let's you define new web-components in your HTML template. It is exposed as a extension of the template tag. In browsers that do not support custom built-ins (the is= syntax), it is triggered via a mutation observer automatically.

Under the hood, it is a wrapper for Tram-Lite's define function. See the documentation there, or the guide on building components, to see all the details about building components.
