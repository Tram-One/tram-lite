tl-controlled

tl-controlled is an attribute for (and similar) tags, that lets you set up 2-way data binding to a parent web-component.

Note, the following example uses Tram-Lite template definitions, but works with any web-component with shadow DOM.

Syntax

Parameters

tl-hostattr optional

Attribute to set on the host web-component, default to "value".

tl-targetattr optional

The attribute to read from the input element, defaults to "value".

tl-trigger optional

The event(s) which should trigger updates to the host component, defaults to "change". Can be multiple values (space delimited).
