tl-controlled

tl-controlled is an attribute for (and similar) tags, that lets you set up 2-way data binding to a parent web-component.

Note, the following example uses Tram-Lite template definitions, but works with any web-component with shadow DOM.

Syntax

Parameters

tl-attrmap optional experimental

Mapping of host attribute names to attribute names of the target element, defaults to "value:value". Can be multiple values (space delimited).

tl-hostattr optional legacy

Attribute to set on the host web-component, default to "value".

tl-targetattr optional legacy

The attribute to read from the input element, defaults to "value".

tl-trigger optional

The event(s) which should trigger updates to the host component, defaults to "change". Can be multiple values (space delimited).

Live Example

<script src="https://unpkg.com/tram-lite@4"></script>

		<template tl-definition>
			<x-greeting>
				<h1>Hello ${'name'}</h1>
			<input tl-controlled tl-hostattr="name" tl-trigger="input" />
			</x-greeting>
		</template>

		<x-greeting name="World"></x-greeting>
