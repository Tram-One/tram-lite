tl-controlled

tl-controlled is an attribute for (and similar) tags, that lets you set up 2-way data binding to a parent web-component.

Syntax

Parameters

tl-attrmap optional

Mapping of target element's attributes to attribute names that will be on the host, defaults to "value:value". Can be multiple values (space delimited).

tl-trigger optional

The event(s) which should trigger updates to the host component, defaults to "change". Can be multiple values (space delimited).

Custom Web Components

If you'd like to use this api with components that aren't defined in Tram-Lite, you can use the Javascript API to apply it to a class using TramLite.appendShadowRootProcessor

Live Example

<script src="https://unpkg.com/tram-lite@5"></script>

		<template tl-definition>
			<x-greeting>
				<h1>Hello ${'name'}</h1>
			<input tl-controlled tl-attrmap="value:name" tl-trigger="input" />
			</x-greeting>
		</template>

		<x-greeting name="World"></x-greeting>
