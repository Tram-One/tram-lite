TramLite.addAttributeListener

TramLite.addAttributeListener is a helper function to set up a callback for when an element's attribute changes. This allows your component to react more dynamically to attribute changes (beyond simple templating).

It is the underlying function behind , and .

Syntax

Parameters

element

Element to watch attribute changes for.

attributeNames

A list of attribute names on the element to watch changes for.

callback

The function to call when one of the listed attributes changes. It has the following parameters:

mutationRecord
A MutationRecord object for the change that was triggered.
