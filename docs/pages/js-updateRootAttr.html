TramLite.updateRootAttr

TramLite.updateRootAttr is a helper function to automatically update an attribute on a web-component when an input value changes. This is a convenience utility that simplifies a common pattern in controlled components.

It is the underlying function behind .

Syntax

Parameters

attributeName

Attribute to set on the host web-component.

event

The default event object for an input component.

targetAttribute optional

The attribute to read from the event.target, defaults to "value"

Return Value

None
