TramLite.appendShadowRootProcessor

TramLite.appendShadowRootProcessor is a utility function used to attach functionality to existing elements that match a CSS selector. It is the underlying function that binds tl-controlled and tl-effect to their elements.

Note, the following example uses Tram-Lite template definitions, but works with any web-component with shadow DOM.

As the name implies, appendShadowRootProcessor works with any element attached to an element's shadowDOM.

Syntax

Parameters

cssSelector

css selectors to match on new elements that are appended to shadowDOM.

behaviorClass

class with a static connect function, which are associated with newly attached nodes.

shadowRootoptional

a specific shadowRoot instance to update the behavior for - by default this is the shadowRoot prototype for all components, but by passing a specific shadowRoot, you only update the behavior of that component's shadowRoot.
