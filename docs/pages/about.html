Tram-Lite

Tram-Lite is an HTML-first library that helps developers build native web-components, and makes building simple native web-applications easier and more elegant!

API

The Tram-Lite API offers the following as part of a small less than 2KB library:

  • A way to build new web-component definitions in HTML
  • Attributes that enhance the behavior for elements made in web-components (regardless of whether those components were defined with Tram-Lite or not)
  • A small set of Javascript functions
  • A way to build your own attributes and enhance other elements consumed by web-components
ComponentDefinition

The recommended way to build custom elements in your HTML template.

define

Template tag function for building native web-components.

addAttributeListener

Function to set up a callback for when an element's attribute changes

updateRootAttr

Event handler to update web-component attributes when inputs change
