Tram-Lite

Tram-Lite is an HTML-first library that helps developers build native web components, and makes building simple native web-applications easier and more elegant!

HTML API

Tram-Lite's main API is through HTML attributes that you can apply to native or custom elements. These attributes (aside from tl-definition) can be used for any web components that use shadow DOM.

tl-definition

Attribute for defining new web components.

tl-controlled

Attribute for establishing 2-way data binding between (and similar) elements and a host web component.

tl-effect

Attribute for building attribute driven side-effects in web components.

tl-rebroadcast

Attribute for emitting events when another event is triggered on an element.

Additionally, Tram-Lite also has a set of custom elements that can be used in web components to enhance their behavior.

tl-broadcast

Custom Element for emitting events when an attribute updates in a web component.

tl-context

Custom Element for pulling state down from a .

tl-provider

Custom Element (can live in the shadow or light DOM), that can provide state to web components with a element.

Javascript API

TramLite.define

Template tag function for building native web components.

TramLite.broadcastEvent

Function to emit events up or down the shadow or light DOM.

TramLite.addAttributeListener

Function to set up a callback for when an element's attribute changes.

TramLite.appendShadowRootProcessor

Build new behaviors for elements that appear in web components.
